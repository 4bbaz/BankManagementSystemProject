# BankManagementSystemProject

<h1 align="center">Bank Management System</h1>

![](/assets/bank.jpg)

Developed a Windows application for bank management, enabling customers to create accounts, perform transactions such as withdrawals and deposits, transfer money, and view their balance statements.

## Built with

- C#
- .NET Framework

## Features

- CRUD.
- Responsive design.
- Information an Account
- Check Balance
